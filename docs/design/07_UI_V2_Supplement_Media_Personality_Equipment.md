## 一、概述

整体框架不推翻。Phase 3、Phase 4 补充三套联动系统：媒体互动、心理人格、装备与赞助联动。最终形成闭环：

```
采访选择 → 塑造公众形象和个性 → 个性与精神属性影响比赛走势
→ 比赛结果改变媒体关注和赞助价值 → 赞助限制装备选择
→ 装备配置影响比赛表现 → 新的比赛结果再次触发媒体事件
```

## 二、媒体采访系统

### 三类采访

| 类型 | 触发条件 | 问题数 |
|------|----------|--------|
| 常规采访 | 普通胜负/排名变化/签约 | 1 题 |
| 重大新闻发布会 | 大满贯/大师赛/ATP决赛/爆冷/首次重要轮次/击败顶尖球员/重伤退赛复出/连败/争议 | 2-4 题 |
| 特殊媒体事件 | 回应赛前言论/公开伤病/解释换教练/回应低迷/赞助商活动/被问No.1或退役 | 1-3 题 |

不能每场比赛都出现新闻发布会。

### 采访触发流程

```
比赛结束 → 赛后结算 → 触发媒体采访 → 玩家选择回答 → 显示即时影响 → 记录进媒体历史
```

未处理采访进入「今日 → 待处理事项」。关键采访未处理时 Continue 可阻止推进。

### 今日 vs 媒体页职责

- **今日**：做采访决定
- **媒体页**：查看公众形象/媒体压力/媒体关系/历史采访/精神属性/个性倾向

## 三、心理人格三层模型

### 1. 精神属性（Mental Attributes — 能力，可成长衰退）

影响比赛实际表现。8 项：

| 属性 | 影响 |
|------|------|
| 镇定 | 破发点/盘点/赛点稳定性 |
| 专注 | 降低领先后掉线/连续 UE |
| 决心 | 落后维持竞争状态，影响逆转概率 |
| 抗压 | 重大赛事和高排名对手面前发挥 |
| 战术理解 | 执行比赛策略、识别对手弱点 |
| 适应力 | 换场地/时差/天气/陌生赛事适应 |
| 预判 | 接发判断、相持判断、网前反应 |
| 情绪控制 | 失分后波动、处罚、连续崩盘风险 |

数值与现有能力统一（1-20 或已有量表）。

### 2. 个性倾向（Personality — 长期人格，非越高越好）

8 项：

| 倾向 | 高值影响 |
|------|----------|
| 职业素养 | 训练执行稳定、恢复自律 |
| 雄心 | 愿挑战高级赛事，但不满足低级赛事 |
| 纪律性 | 战术执行严格，较少脱离计划 |
| 侵略倾向 | 强势比赛风格，可能引发争议 |
| 冒险倾向 | 关键分主动进攻，也可能增加失误 |
| 体育精神 | 公众好感，减少争议处罚 |
| 媒体开放度 | 提升人气，也承受更多曝光压力 |
| 商业倾向 | 看重赞助收益，影响合同选择 |

人格变化极其缓慢——不会回答一次采访就改变。

### 3. 当前心理状态（Short-term State）

信心 / 压力 / 士气 / 情绪 / 比赛紧张度

### 三层关系

```
个性倾向  → 决定球员通常如何反应
精神属性  → 决定他有多大能力控制和执行
当前心理状态 → 决定这一场比赛当下处于什么状态
```

案例：侵略高+镇定低+压力高 → 赛点冒险强攻并失误。侵略高+镇定高 → 关键分主动制胜。

## 四、比赛模拟整合

比赛走势公式：

```
基础实力 × 场地与对位 × 体能状态 × 当前心理状态 × 精神属性修正 × 比赛事件波动
```

个性不提供直接能力加成，而决定事件发生后的反应路径。关键影响环节：关键分表现/落后反应/领先稳定/失误后波动/战术执行/高压赛事发挥/主场与媒体压力适应/伤愈复出状态。

## 五、采访选择影响模型

三条规则：
1. 每次选择只明显改变短期状态，长期人格极小变化
2. 同类选择长期重复才逐渐形成性格标签
3. 选项无绝对正确答案——代价不同

示例：

> 记者："你是否认为自己已经具备争夺冠军的能力？"

| 选择 | 即时影响 | 长期倾向变化 |
|------|----------|-------------|
| "我来这里就是为了夺冠。" | 信心+3 / 媒体关注+2 / 媒体压力+2 | 雄心微增 / 谦逊微降 |
| "我只专注于下一场比赛。" | 专注+2 / 媒体压力-1 | 职业素养微增 |
| "签表中的每个人都有机会。" | 公众好感+1 | 体育精神微增 / 信心不变 |

## 六、装备系统扩展

### 装备拆分为五槽位

```
球拍 → 基础定位与长期属性加成（控制/力量/旋转/稳定/甜区/网前）
球线 → 细致短期比赛体验（旋转/手感/控制/弹性/耐用/舒适）+ 类型 + 磅数
球鞋
球衣与服装
配件（未来）
```

### 磅数影响

- 高磅：控制↑ / 力量↓ / 舒适↓
- 低磅：力量↑ / 甜区↑ / 控制↓

装备提供微调，不决定胜负。

## 七、品牌赞助与装备联动

### 合同槽位（6 个）

主赞助 / 球拍赞助 / 服装赞助 / 球鞋赞助 / 球线赞助 / 地区合作

### 页面职责分离

| 页面 | 职责 |
|------|------|
| **装备页** | 当前穿着使用/属性/外观预览/更换配置 |
| **赞助页** | 品牌独家要求/合同收入/奖金条件/期限/品牌限制/违约后果 |

### 联动规则

签约 Nike 服装赞助后：
- 球衣鞋可选品牌受限
- 免费/折扣获得一定等级装备
- 每周固定收入 + 大赛晋级/夺冠奖金
- 穿错品牌触发警告或违约

签约 Wilson 球拍后：
- 只能用 Wilson 合同允许系列
- 品牌免费提供升级/新品
- 更换品牌需先解约
- 真实比赛表现影响续约报价

### 核心原则

**赞助合同不自动把装备变成最强。** 高额赞助提供的装备未必最适合——形成三角决策：

```
更适合自己的装备  vs  更高赞助收入  vs  合同限制
```
