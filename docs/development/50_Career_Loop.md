# Overview

Career Loop 定义整个 Ace Career 的核心游戏循环。

前面所有 Engine（Training/Match/Recovery/Story/Ranking/Media/Sponsor/Legacy）最终都会汇聚到这里。如果说 Match Engine 是心脏，Career Loop 就是整个身体的血液循环。

---

# Design Goals

✓ 简单易懂 ✓ 长期可玩 ✓ 每周反馈 ✓ 每赛季成长 ✓ 每阶段有目标 ✓ AI/玩家一致 ✓ 百年不疲劳

---

# 五层循环

```
Daily (1~3 分钟) → Weekly → Tournament → Season → Career → Legacy
```

---

# Core Gameplay Loop

```
Today → Train → Recover → Tournament → Match → Ranking → Story → Career Growth → Next Week
```

---

# 正向/负向循环

- **Positive**：Win→Confidence→Popularity→Sponsors→Better Staff→Better Growth
- **Negative**：Lose→Recovery→Training→Comeback→Story（失败也是职业的一部分）

# 关键设计

- **反馈立即**：Training→Growth→Match Win→Ranking Up→News→Achievement
- **目标动态**：系统根据职业阶段自动推荐，玩家无需自己寻找
- **AI 一致**：AI 同样遵循 Train→Compete→Recover→Improve→Retire
- **Story 持续**：每个循环产生 Story→Narrative→News→Timeline→Legacy
- **Economy 循环**：Results→Sponsors→Income→Staff→Training→Better Results
- **情绪循环**：Hope→Pressure→Victory→Failure→Comeback→Legacy
- **Endgame**：No.1 之后仍有 Defend Legacy→Break Records→Hall Of Fame→GOAT Debate
- **退休不是结束**：Hall Of Fame→Career Documentary→GOAT Ranking→History Archive → 重新开始
- **世界永续**：玩家退役后 AI 继续成长/新球员出现/新传奇诞生

# Final Principle

Ace Career 真正想模拟的不是一场网球比赛，而是一位职业球员的一生。

冠军只是旅程中的一个站点。真正值得追求的，是当几十年以后，整个职业网球世界依然会记住你的名字。
