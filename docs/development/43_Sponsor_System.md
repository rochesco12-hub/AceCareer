# Overview

Sponsor System 定义整个 Ace Career 的商业赞助系统。

赞助不是奖励，而是一种认可。真正的顶级职业球员收入不仅来自奖金，更来自品牌价值。品牌不会只签约世界第一——他们更关心全球影响力、市场价值、国家影响、人设、年龄、未来潜力。

---

# Key Features

- **Brand Prestige**：Local ★1 → Global ★5
- **Brand Preference**：Young Talent→Nike / Legend→Rolex / Power Player→Wilson
- **Multiple Sponsors**（未来）：Racket(Wilson) + Shoes(Nike) + Watch(Rolex) 互不冲突
- **Contract Renewal**：合同到期品牌重新评估 Results/Popularity/Reputation → Renew or Terminate
- **Sponsor Loss**：Popularity↓ → Contract Ends → Revenue↓（职业低谷商业价值同步下降）
- **Exclusive Contracts**（未来）：Nike × Adidas 品牌独占
- **Media Effect**：重大签约自动生成 News→Story→Career Timeline
- **Financial Impact**：赞助收入进入 Career Income→Economy→Net Worth
- **AI Sponsorship**：AI 同样签约，查看 AI 球员可见品牌合作
