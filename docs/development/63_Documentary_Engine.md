# Overview

Documentary Engine 定义整个 Ace Career 的纪录片系统。

History 保存历史。Replay 回顾历史。Documentary 重新讲述历史。真正优秀的体育纪录片不会展示每一场比赛，而会选择最重要的故事、最重要的人、最重要的时代。Documentary 不是历史年鉴，而是一段人生。

✓ 全自动生成 ✓ Story First ✓ AI/玩家一致 ✓ Config Driven ✓ 不依赖 AI 幻觉 ✓ 永久保存 ✓ 百年稳定

# Key Features

- **Career Documentary**：Beginning→Rise→Peak→Challenges→Legacy→Retirement（完整人生）
- **Season Documentary**：Highlights/Titles/Ranking/Stories/Awards（每赛季自动回顾）
- **Tournament Documentary**：Road To Champion/Upsets/Final/Champion（完整冠军之路）
- **Rivalry Documentary**：First Meeting/Classic Battles/Grand Slam Finals/Last Match
- **Era Documentary**：Era Begins/Dominance/Challenges/Transition
- **Hall Of Fame Documentary**：Biography/Timeline/Legacy/Ceremony（最终人物传记）
- **World Documentary**（每十年）：Greatest Players/Matches/Stories/Historic Moments
- **Top 20 Moments**：Story Weight + Narrative + Historic Value 自动筛选
- **AI Narration**（未来）：自动生成 Narration/Biography/Chapter Summary
- **Visual Timeline**：🏆 First Grand Slam→👑 World No.1→🏛 Hall Of Fame（Apple 风格）
- **Documentary Assets**（未来）：Photos/Match Posters/Career Cards/Trophies/Statistics
- **Interactive Documentary**：点击 Story→Replay→Statistics→Timeline 自由探索
- **Share Documentary**（未来）：导出 Career Album/Story/Season Review
- **AI Documentary**：所有 AI 退役后自动拥有纪录片
